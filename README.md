# 🚀 TravelPlanner

A simple iOS app for planning trips and organizing itineraries in one place.

---

## ✨ Current Focus

- View a list of trips  
- Navigate into a trip’s itinerary *(in progress)*  
- Build core navigation and data structure  

---

## 🛠 Tech Stack

- Swift  
- UIKit (Storyboard-based UI)  
- UITableView  
- Auto Layout  

---

## 🚧 Status

Currently building the core foundation of the app:

- Trips list screen (in progress)
- Itinerary screen (next)
- Navigation between screens
- Local data persistence

---

## 📱 Planned Features (V1)

- Create and manage multiple trips  
- Add itinerary items (title, time, location, notes)  
- View itinerary in a list  
- Open locations in Apple Maps  
- Map view with pins  

---

## 🌱 Future Direction

- Collaborative trip planning with friends  
- Shared itineraries and real-time updates  
- AI-assisted planning (suggestions, packing lists, itinerary optimization)  

---

## 🧠 About

TravelPlanner aims to solve the problem of scattered travel information by providing a centralized, mobile-first planning experience. The long-term vision is to make trip planning collaborative and intelligent, while keeping the experience simple and intuitive.
